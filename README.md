# Cracking the Interview - Resume Repository

Welcome to the official repository for uploading resumes as part of the "Cracking the Interview" initiative. This project is built using [Next.js](https://nextjs.org/), a React framework that enables functionality such as server-side rendering and static site generation.

## About

The main goal of this repository is to create a platform where users can upload their resumes for mentors in the tech industry to review. It's part of our mission to bridge the gap between fresh grads, developers, and top tech companies.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) 12.x or later
- [npm](https://www.npmjs.com/) 6.x or later or `pnpm`

### Installation

1. Clone this repository:

`git clone https://github.com/xxxoooxoxo/cti-resume-upload`

2. Navigate to the project directory:

`cd cti-resume-upload`

3. Install the project dependencies:

`npm install`

4. Run the development server:

`npm run dev`

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
